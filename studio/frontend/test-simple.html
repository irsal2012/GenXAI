This is a simple HTML page with similar styling to test if the issue is with React or CSS.

Test Card 1

If you see a large "M" logo on this page too, the issue is NOT with the React app.

It might be a browser extension, system overlay, or another application.

Test Card 2

If you DON'T see the "M" logo on this page, then the issue is specific to the React application.

Debugging Steps

  • Try opening in an incognito/private window
  • Disable all browser extensions
  • Check if any other applications are running that might overlay graphics
  • Try a different browser
